# StatsGate limitations

- Checks **arithmetic/statistical consistency only**. It cannot detect fabrication that happens to be internally consistent — a fabricated dataset can pass every check here. It does not replace a real data audit, a statistician's review, or access to the raw data.
- **GRIM** assumes the mean comes from a fixed number of **integer-scored** items (e.g. a Likert scale, a count). It does not apply to continuous measurements (height, reaction time in ms, etc.) unless those are also integer-granular.
- **SD range**'s minimum bound is the exact integer optimum (equal-as-possible split) and is always correct. Its **maximum bound is a continuous-relaxation upper bound** — it solves the bounded-variance-maximization problem allowing one real-valued "patch" element instead of enforcing every element to be an integer. This means the true integer-constrained maximum achievable SD is always `<=` the bound reported here. Consequence: StatsGate can only ever **under-flag** on the high side (an SD just above the true integer maximum but still under this looser bound will be reported `NOT_RULED_OUT`, not `IMPOSSIBLE`) — it will never wrongly call a truly-achievable SD impossible.
- `NOT_RULED_OUT` is not `CONFIRMED` — it means the reported SD falls within the achievable range, not that a matching dataset has been shown to exist (unlike full SPRITE, this tool does not perform an iterative search for an actual matching integer sequence).
- **P-value check** covers **Student's t-tests only** (two-tailed), via the exact regularized incomplete beta function. F-tests, chi-square tests, and one-tailed tests are out of scope in this lite version.
- Tolerance for the p-value check defaults to 0.005 to allow for reporting rounding (e.g. a paper reporting "p = .04" for a computed p = .038); tighten it if you need stricter matching.
- No dependencies beyond the Python standard library — the t-distribution CDF is a from-scratch implementation of a well-known numerical algorithm (Lentz's continued fraction for the incomplete beta function), verified in this repo's tests against textbook critical-value tables, not against a reference statistics library.
